# ROS 2 Installation

## Overview

This guide covers the installation of ROS 2 Humble Hawksbill on Ubuntu 22.04 LTS, the recommended platform for this course.

## Prerequisites

- Ubuntu 22.04 LTS (Jammy Jellyfish)
- Minimum 4GB RAM (8GB recommended)
- 20GB free disk space
- Stable internet connection

## Installation Steps

### 1. Set up Sources

```bash
# Enable Ubuntu Universe repository
sudo apt install software-properties-common
sudo add-apt-repository universe

# Add ROS 2 GPG key
sudo apt update && sudo apt install curl -y
sudo curl -sSL https://raw.githubusercontent.com/ros/rosdistro/master/ros.key -o /usr/share/keyrings/ros-archive-keyring.gpg

# Add repository to sources list
echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/ros-archive-keyring.gpg] http://packages.ros.org/ros2/ubuntu $(. /etc/os-release && echo $UBUNTU_CODENAME) main" | sudo tee /etc/apt/sources.list.d/ros2.list > /dev/null
```

### 2. Install ROS 2

```bash
# Update package index
sudo apt update
sudo apt upgrade

# Install ROS 2 Humble Desktop
sudo apt install ros-humble-desktop

# Install development tools
sudo apt install ros-dev-tools
```

### 3. Environment Setup

```bash
# Source ROS 2 setup in your bashrc
echo "source /opt/ros/humble/setup.bash" >> ~/.bashrc
source ~/.bashrc

# Verify installation
ros2 doctor
```

## Verification

Test your installation:

```bash
# Terminal 1: Start talker
ros2 run demo_nodes_cpp talker

# Terminal 2: Start listener
ros2 run demo_nodes_py listener
```

You should see messages being published and received.
