# Module 3: AI-Robot Brain (NVIDIA Isaac) Examples

This directory contains runnable examples that demonstrate the AI concepts covered in Module 3 of the Physical AI & Humanoid Robotics Book. These examples illustrate AI pipeline concepts, perception systems, navigation algorithms, and learning approaches specifically designed for humanoid robots using NVIDIA Isaac.

## Directory Structure

```
examples/
├── README.md                    # This file
├── ai_pipeline_basics/          # Basic AI pipeline components
├── perception_examples/         # Visual perception and SLAM examples
├── navigation_examples/         # Bipedal navigation implementations
├── learning_examples/           # Machine learning and sim-to-real examples
└── isaac_integration/           # Isaac-specific integration examples
```

## Examples Overview

### 1. AI Pipeline Basics
- Basic perception pipeline with sensor fusion
- Real-time performance optimization
- Modular component design

### 2. Perception Examples
- Visual SLAM implementation
- Feature detection and tracking
- Multi-modal sensor fusion

### 3. Navigation Examples
- Bipedal path planning
- Footstep planning algorithms
- Balance-aware navigation

### 4. Learning Examples
- Reinforcement learning for locomotion
- Imitation learning implementations
- Sim-to-real transfer techniques

### 5. Isaac Integration
- Isaac ROS package usage
- GPU-accelerated processing
- TensorRT optimization examples

Each example includes:
- Complete source code
- Configuration files
- Usage instructions
- Performance benchmarks
